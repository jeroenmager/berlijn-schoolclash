# Schoolclash application


Dit is een verbetering van de Schoolclash applicatie gemaakt door de amateuristische 3e jaars van 2017/2017


Getting started React-Native:

- https://facebook.github.io/react-native/docs/getting-started.html


Versies:
  
- react-native -v
    
- react-native-cli: 2.0.1
    
- react-native: 0.43.2
  
- npm -v
    
- 5.0.4


In AndroidStudioProjects:

- Creëer een map met de naam SchoolClash

- Open Git Bash

- Git init

- Git remote add origin https://www.github.com/jeroenmager/berlijn-schoolclash

- Git pull origin master


Update alle packages (npm):

- npm install


Opstarten van de applicatie applicatie:

- Open CMD

- In CMD typ React-native run-android

- Wanneer je BatchedBrigde error krijgt, open een nieuwe CMD en typ in React-native start


Zorg ervoor dat je GSM in developer mode staat (Wanneer je de applicatie wilt builden op GSM):

- Wanneer je niet weet hoe dit moet, Google it.
